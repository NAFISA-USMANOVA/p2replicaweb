### General Info
***
The individual 5-day sprint project. The purposes of this training project are programming a responsive web page for the 'mobile' size based on HTML and CSS, which includes structuring the Page, using frameworks, git, deploying the project on Gitgub pages, Netify or another server. 

## Technologies
***
A list of technologies used within the project:
* HTML5
* CSS3
* w3schools (https://www.w3schools.com)

## Requirements
***
* Elaborate an exact replica of the mockups delivered, complying
with: the color palette, the font and the iconography
* The page must be responsive
* The navbar should show where on the page the user is, changing the
color
* Buttons must be clickable
* The Download button must take you to the factory page
* Element sizes must be the same as the mockup elements
* The mobile of the “Mobile app landing page” section must have an animation
* When the user hovers over a button you MUST see a glow effect
* When a user hovers over an icon, the icon MUST be animated


## Done
***
* Basics of Front End
* Home page (static navbar, animated image)
* Clickable Buttons
* The Download button does take a user to the FactoriaF5 page
* Links
* When a user hovers over an icon, the icon is animated
* The project is deployed to Github
